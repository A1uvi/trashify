Trashify is the Solution:
It filters out cardboard, glass, metal, paper, and plastic out from general trash to save wildlife in the ocean and on land.

How it Works

Data Preprocessing using Python

I did data processing in python by transforming, augmenting and analyzing the 2527 images in my dataset so that all of the values became usable and properly recognized.

Deep Learning using Tensorflow

CNN (Convolutional Neural Network) is a deep learning keras algorithm in Python used for image classification. It involved filtering an image through multiple layers, in this case 12, and iterating through the data 100 times to identify patterns.

User Experience using HTML, CSS, JavaScript

HTML provides the structure, CSS adds styling and visual appeal, while JavaScript adds interactivity and dynamic behavior to a website, creating a seamless and engaging user experience.

Full Stack Framework using Flask

Flask allowed me to use python functions in the backend to capture and save images to then classify with my loaded model. It also helped me to connect my HTML to the outputs of the python functions.
